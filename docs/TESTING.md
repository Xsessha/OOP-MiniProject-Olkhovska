## У даній ітерації реалізовано тестування бізнес-логіки системи оренди автомобілів.

## Огляд
Цей проєкт використовує модульне та інтеграційне тестування для забезпечення коректності системи оренди автомобілів.

## Типи тестів
- Модульні тести: доменна логіка (Car, Rental, RentalService)
- Інтеграційні тести: повний процес оренди

## Інструменти
- xUnit
- .NET Test SDK

## Що тестується
- Оренда автомобіля
- Повернення автомобіля
- Граничні випадки (некоректний ID, відсутній автомобіль)
- Запобігання повторній оренді
