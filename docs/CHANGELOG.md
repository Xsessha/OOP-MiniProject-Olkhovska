- Створено доменний рівень (Domain layer)
- Додано основні сутності предметної області:

**Результат інкременту:**

## lab 35 — Формування бізнес-логіки та архітектури шарів

- Додано сервісний шар:
  - `RentalService`
- Впроваджено бізнес-логіку оренди та повернення автомобілів
- Реалізовано патерн **Repository**:
  - відокремлення логіки доступу до даних від бізнес-логіки
- Додано in-memory збереження даних
- Створено базовий консольний інтерфейс (Menu + handlers)

**Архітектурні покращення:**
- введено розділення на шари (Domain / Application / Infrastructure / UI)
- застосовано DIP через залежність від абстракцій

**Результат інкременту:**
- працюючий вертикальний slice: оренда автомобіля end-to-end
