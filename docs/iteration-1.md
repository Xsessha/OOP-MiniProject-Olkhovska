# Iteration 1

## Реалізовано
- Базова бізнес-логіка оренди авто (Domain та Application шари).
- Інтерактивний консольний інтерфейс (CLI) для взаємодії з користувачем.

## Артефакти
- **UML:** Створено архітектурні схеми ([Class Diagram](class-diagram.md) та [Sequence Diagram](sequence-diagram.md)).
- **Tests:** Написано unit-тести (xUnit) для перевірки сервісу оренди.
- **CI/CD:** Налаштовано GitHub Actions для автоматичної збірки та перевірки тестів.

## Далі 
- **Persistence:** Збереження стану програми (щоб дані не втрачалися після закриття консолі).
- **LINQ:** Використання LINQ-запитів для фільтрації, пошуку та сортування списку автомобілів.
