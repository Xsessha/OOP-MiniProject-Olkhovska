# Система оренди автомобілів — Посібник користувача

## Огляд
Ця система — це консольний застосунок для оренди автомобілів, який дозволяє користувачам:
- Переглядати доступні автомобілі
- Орендувати автомобіль
- Повернути автомобіль

Система спроєктована на основі багаторівневої архітектури (Domain, Application, Infrastructure, Presentation).

##  Як запустити застосунок

### Вимоги
- .NET 8-9 SDK або новішої версії
- Консоль/Термінал

### Кроки для запуску
```bash
dotnet build
dotnet run --project src/MyProject.Console
```

### Головне меню
Після запуску застосунку ви побачите:

1. Орендувати автомобіль
2. Повернути автомобіль
3. Показати доступні автомобілі
0. Вийти

##  Варіанти використання

### 1. Оренда автомобіля

Оберіть опцію 1

Введіть своє ім'я

Введіть ID автомобіля
Якщо успішно:

Успішно орендовано!

 Якщо помилка:

Автомобіль вже орендовано АБО недійсний ID

### 2. Повернення автомобіля

Оберіть опцію 2

Введіть своє ім'я

Введіть ID автомобіля
Успіх:

Успішно повернуто!

### 3. Перегляд доступних автомобілів
Відображає всі автомобілі зі статусом:

ID  Модель  Доступний/Орендований


## Обробка помилок

Недійсний формат GUID → відхилено

Порожнє ім'я - відхилено

Оренда вже орендованого автомобіля - повідомлення про помилку

## Зберігання даних

У поточній версії використовується зберігання в оперативній пам'яті

Легко замінюється на збереження у файли (JSON)

## Тестування
Всі основні процеси покриті тестами:

Успішна/невдала оренда автомобіля

Логіка повернення автомобіля

Негативні сценарії
